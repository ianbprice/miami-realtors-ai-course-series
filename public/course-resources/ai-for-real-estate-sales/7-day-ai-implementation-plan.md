# 7-Day AI Implementation Plan for Real Estate Agents

Course: AI for Real Estate Sales: From Confusion to Daily Use

Use this plan to turn the class into one repeatable habit. Keep the first week simple: one workflow, one prompt, one review habit.

## Day 1: Pick One Workflow

Goal: Choose where AI will help first.  
Task: Select one repeated task from your normal week.  
Tool/prompt to use: Daily task planning prompt from the prompt library.  
Expected outcome: One practical workflow to test.  
10-minute action step: Write down three tasks you delay because they are repetitive, awkward, or time-consuming. Pick one.

## Day 2: Build Your First Reusable Prompt

Goal: Turn the workflow into a repeatable prompt.  
Task: Use the real estate prompt formula.  
Tool/prompt to use: Role, task, context, audience, tone, constraints, format, and goal.  
Expected outcome: One saved prompt you can reuse.  
10-minute action step: Write the prompt with placeholders such as [CLIENT NAME], [LEAD SOURCE], [PROPERTY FACTS], and [NEXT STEP].

## Day 3: Test It On A Low-Risk Draft

Goal: Practice without exposing private information.  
Task: Use synthetic or sanitized notes to generate a draft.  
Tool/prompt to use: Email follow-up or text message follow-up prompt.  
Expected outcome: A usable draft and a list of edits.  
10-minute action step: Run the prompt once, then mark what you would change before sending.

## Day 4: Add Your Review Checklist

Goal: Create a review habit before client use.  
Task: Check the output for accuracy, privacy, tone, and compliance issues.  
Tool/prompt to use: Compliance review checklist prompt.  
Expected outcome: A safer, reviewed version.  
10-minute action step: Ask AI to review its draft for risk, then make your own final judgment.

## Day 5: Apply It To A Real Business Task

Goal: Use AI on a real task with appropriate safeguards.  
Task: Draft a lead follow-up, seller update, buyer follow-up, listing asset, or appointment prep outline.  
Tool/prompt to use: The prompt that matches your selected workflow.  
Expected outcome: One reviewed business asset.  
10-minute action step: Use verified facts and remove confidential details before prompting.

## Day 6: Save What Worked

Goal: Build a small personal prompt library.  
Task: Save the original prompt, improved prompt, final output, and notes.  
Tool/prompt to use: Repeatable prompt workflow.  
Expected outcome: One reusable workflow template.  
10-minute action step: Create a folder or note named "AI prompts that worked" and save the prompt.

## Day 7: Measure And Improve

Goal: Decide whether the workflow helped.  
Task: Review time saved, quality, and follow-up results.  
Tool/prompt to use: Daily task planning or lead follow-up measurement prompt.  
Expected outcome: One improvement for next week.  
10-minute action step: Answer: Did this help me draft faster, communicate more clearly, or follow up more consistently?

## Week-One Success Standard

You do not need a complete AI system by the end of the week. You need one repeatable workflow that saves time, improves clarity, and still keeps you responsible for the final result.
